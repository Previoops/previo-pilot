# Previo Pilot

Previo helps engineering teams stop shipping blind by surfacing relevant prior incidents, constraints, and deployment context before releases go live.

Previo runs inside your CI/CD workflow and is advisory-first.

## What this repo is

This repository contains pilot integration examples, setup guides, and security information for teams evaluating Previo.

It does **not** contain Previo source code.

## Quick Start

1. Add the GitHub Actions example from `examples/github-actions/previo-check.yml`
2. Add your Previo pilot API key as a GitHub secret
3. Run your pipeline
4. Review Previo advisories in CI logs
